# Course Management Web App

### Tech Stack

- Backend
  - maven
  - java
  - Spring boot
  - MySQL
- Frontend
  - ReactJS

### Functional Requirement

- Admin Panel
  - add courses
    - no of weeks
    - price
    - duration
    - student data for course
- User Panel
  - buy course
  - course visible in panel

### Non Functional Requirement

- Security
- Performance
- Scalability
- Usability
- Reliability
